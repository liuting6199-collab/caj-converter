# 🚀 快速开始指南

## 最简单的使用方法

### 第一步：双击启动
直接双击 `start.bat` 文件！

### 第二步：打开浏览器
访问：http://127.0.0.1:8000

### 第三步：上传文件
拖拽CAJ文件到网页，或点击选择文件

### 第四步：下载PDF
等待转换完成，点击"下载"按钮

---

## 📱 创建桌面快捷方式

1. 右键 `start.bat`
2. 选择"发送到" → "桌面快捷方式"
3. 以后只需双击桌面图标即可！

---

## 🔧 如果start.bat不行

### 安装Python依赖
打开命令行，进入项目目录：
```bash
cd e:\2026edu\caj_converter
pip install -r requirements.txt
```

### 手动启动
```bash
python main.py
```

然后打开浏览器访问 http://127.0.0.1:8000

## ❓ 常见问题

**Q: 提示找不到命令？**
A: 请先安装Python：https://www.python.org/downloads/

**Q: 转换失败怎么办？**
A: 确保文件是CAJ/KDH/NH格式，文件大小不超过50MB

**Q: 可以在手机上用吗？**
A: 目前只能在电脑上使用，需要Python环境
